# Employee Tracker
  
  ## Description
  This content management system serves as an interface through which non-developers can view and interact with an employee database. This CMS uses Node.js, an npm package called Inquirer, and PostgreSQL. It allows users to view all the employees at a company as well as employee roles, salaries, and departments.

  ## Installation
  1.) Clone the repo. https://github.com/mancillh/SQL-Employee-Tracker
  2.) Use "npm i" in the terminal to install dependencies. 
  3.) Set up the schema and seed the databases by typing "psql -U" followed by a space and your username. When prompted, enter your Postgres password. Into the terminal, type "\i db/schema.sql", enter, then "db/seeds.sql", enter, then finally "\q". 
  4.) Type "npm start" into the terminal. 
  5.) Make your selections to view, add, or update as desired. Enjoy!

  ## Usage
  This CMS may be used to view all departments, view all roles, view all employees, add a department, add a role, add an employee, and update an employee role.
